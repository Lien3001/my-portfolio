# my-portfolio
## Elements
My vritual CV has 4 pages:
- Home
- Resume
- Projects
- Contact
## Features
Each of these pages has a navigation bar that allows the user to navigate between them. The navigation bar collapses when the screen is too small to  contain all the content and offers  a burger menu instead.\
All pages use bootstrap css as well as a custom style sheet written for the prohect.\
There is a typing animation on the home page that allows the user to see multiple roles that I can fulfill.\
The website effectively uses boostrap and flexbox to have responsive design on all viewports.
## The flow
### Home page
The home page starts with a short introduction to me, including a photo and typing animation that shows all possible jobs that I can do. When scrolling down, links to all the other pages (CV, projects, contact) are made available with enticing messages to each one.
### Resume
The resume page is a virtual CV that includes Education, Awards, Certifications, Experience, Technical Skills, Creative software and languages. It also includes a download link to a pdf version  of my cv.
### Projects
The projects page outlines 3 projects and links to repositories on GitHub as well as a link to my github profile.
### Contact
The contact page includes all information that a future employer will require to hire me.
